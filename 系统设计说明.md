# PyBullet 多模态抓取系统设计说明

## 1. 项目目标

本项目基于 PyBullet 搭建 Panda 机械臂抓取仿真系统，目标是实现：

- 视觉感知定位目标物体
- 压力感知判断夹持稳定性
- 惯性感知评估运输过程动态扰动
- 多模态融合形成抓取状态
- 闭环控制根据融合结果自动调整夹爪
- 完成抓取、抬升、搬运、放置全流程

---

## 2. 当前系统架构

核心代码位于 `grasp_system/`，模块职责如下：

### `config.py`
统一管理系统参数，包括：

- 场景参数：机械臂、物体、随机生成范围
- 运动参数：抓取高度、抬升高度、姿态候选、搜索半径
- 多模态融合参数：置信度阈值、风险阈值
- 闭环控制参数：加力步长、夹爪收紧步长、放置阶段保护阈值

### `simulation.py`
负责：

- 连接 PyBullet
- 加载地面、机械臂、物体
- 随机生成物体位置
- 维护仿真步进与回调

### `vision.py`
负责视觉定位：

- 从相机深度/图像中估计物体位置
- 输出视觉坐标
- 与仿真真值对比记录误差日志

### `imu.py`
负责记录：

- 末端执行器速度/加速度
- 物体速度/加速度

主要用于评估运输阶段是否存在滑移或动态扰动。

### `pressure.py`
负责统计夹爪与物体接触信息：

- 总法向力
- 总切向力
- 左右指法向力
- 接触手指数
- 稳定接触比例

### `multimodal.py`
负责多模态融合，输出：

- `state`
- `grasp_confidence`
- `slip_risk`
- `vision_confidence`
- `pressure_confidence`
- `imu_stability_score`

并生成 `alerts`，如：

- `lateral_ratio=...`
- `object_motion=...`
- `low_grasp_confidence=...`

### `robot.py`
负责机械臂控制：

- IK 求解
- 姿态候选生成
- yaw / roll / pitch 抓取姿态搜索
- 到位验证

### `gripper.py`
负责夹爪控制：

- 张开/闭合
- 保持夹持
- 搬运强化夹持
- 自适应收紧

### `workflow.py`
系统总流程控制器，负责：

- 回家位
- 视觉锁定
- 候选抓取姿态尝试
- 验证抬升
- 闭环控制
- recenter 回正
- 搬运
- 放置

## 3. 抓取流程主线

当前完整流程如下：

1. **初始化与回家位**
   - 机械臂移动到 home
   - 记录 `after_home`

2. **视觉锁定目标**
   - 视觉模块输出物体位置
   - 记录 `VISION_SUMMARY`
   - 记录 `after_vision_lock`

3. **抓取姿态规划**
   - 根据目标位置生成多个候选姿态
   - 同时搜索：
     - 多个 yaw
     - 少量 roll / pitch 倾斜姿态
     - 小范围 XY 偏移

4. **执行抓取**
   - 先移动到 `pre_grasp`
   - 再下降到 `grasp_pos`
   - 关闭夹爪
   - 记录 `after_close_attempt_x`

5. **验证抬升**
   - 小幅抬升，判断：
     - 物体是否离地
     - 物体是否仍在末端附近
   - 记录 `after_validation_lift_attempt_x`

6. **recenter 回正**
   - 若检测到夹持中心偏移，则微调末端 XY
   - 记录 `GRASP_RECENTER`
   - 记录 `after_recenter_attempt_x_y`

7. **正式抬升与搬运**
   - 强化夹持
   - 抬升到运输高度
   - 搬运到放置区域
   - 记录 `after_lift`

8. **放置**
   - 移动到 `pre_place`
   - 下放到 `release`
   - 打开夹爪
   - 后撤并回 home
   - 记录 `before_release / after_release / PLACE_FINISH`

## 4. 多模态融合思路

### 4.1 视觉模态
作用：

- 抓取前定位目标
- 提供目标与规划位置之间的误差参考

特点：

- 抓取前权重高
- 运输阶段冻结视觉置信度，不让视觉误差错误干扰搬运判断

### 4.2 压力模态
作用：

- 判断是否真的夹住
- 判断左右受力是否均衡
- 判断切向力是否过大

关键指标：

- `pressure_normal`
- `pressure_balance`
- `pressure_lateral_ratio`
- `pressure_confidence`

### 4.3 惯性模态
作用：

- 判断运输过程中物体是否晃动
- 判断是否存在滑移趋势

关键指标：

- `object_speed`
- `object_peak_acc`
- `ee_peak_acc`
- `imu_stability_score`

### 4.4 融合输出
最终融合得到：

- **抓取置信度** `grasp_confidence`
- **滑移风险** `slip_risk`
- **融合状态** `state`

常见状态包括：

- `HOMED`
- `VISION_LOCKED`
- `ALIGNING_FOR_GRASP`
- `CONTACT_ESTABLISHED`
- `GRASP_STABLE`
- `TRANSPORT_STABLE`
- `TRANSPORT_UNCERTAIN`
- `SLIP_RISK`
- `RELEASED`

## 5. 闭环控制设计

### 5.1 基本思想
闭环控制不是简单“夹紧到底”，而是：

1. 监测多模态融合状态
2. 如果出现不稳定趋势，则轻量补偿
3. 如果发现偏心抓取，则进行回正
4. 如果抓取质量仍不够好，则放弃该姿态，切换下一个候选

### 5.2 当前闭环动作

#### 动作 1：夹爪自适应收紧
当满足以下情况时触发：

- `slip_risk` 偏高
- `pressure_confidence` 偏低
- `grasp_confidence` 偏低
- 状态为 `TRANSPORT_UNCERTAIN / SLIP_RISK`

执行内容：

- 增大夹持力
- 略微减小夹爪目标开度

对应日志：

- `GRIP_ADAPTIVE`
- `FUSION_CONTROL`

#### 动作 2：抓取回正 recenter
当满足以下情况时触发：

- 验证抬升后物体在末端 XY 上仍有偏差
- 或受力显示仍有偏心趋势

执行内容：

- 末端小幅 XY 微调
- 使物体尽量回到夹爪中部

对应日志：

- `GRASP_RECENTER`
- `after_recenter_attempt_*`

#### 动作 3：抓取质量门控
即使物体被提起，也要检查：

- `grasp_confidence`
- `slip_risk`
- `pressure_lateral_ratio`

如果质量不达标，则本次抓取失败，换姿态重试。

对应日志：

- `GRASP_QUALITY_REJECT`

### 5.3 放置阶段保护
为避免 pre-place/release 阶段过控，当前已加入：

- 放置阶段稳定态豁免
- 放置阶段最大动作次数限制

这样可以避免系统在已经稳定时继续无意义收紧夹爪。

## 6. 当前系统效果总结

### 已取得效果

1. **视觉定位稳定**
   - 当前误差已基本稳定在毫米级到几毫米级

2. **抓取成功率明显提高**
   - 通过候选姿态 + 多模态门控，抓取稳定性提升明显

3. **掉落问题已基本解决**
   - 通过运输阶段夹持强化与闭环补偿，物体不再容易在搬运时脱落

4. **偏心抓取得到改善**
   - recenter 已能有效减小物体相对末端的 XY 偏差

5. **闭环已从“能用”提升到“较稳定”**
   - 不再主要依赖暴力夹紧
   - 已具备一定“检测-补偿-纠偏”能力

### 当前仍存在的限制

1. 某些极限位置仍可能超出工作空间
2. 个别情况下运输阶段仍偏保守
3. recenter 目前只做 XY 微调，尚未进行姿态级纠偏
4. 当前物体类型较简单，尚未扩展到复杂形状

## 7. 日志系统说明

当前日志已经比较完整，可用于论文实验分析。

### 关键日志类型

- `VISION_SUMMARY`
  - 视觉位置与真值误差

- `FUSION_STATE`
  - 当前融合状态与核心指标

- `FUSION_CONTROL`
  - 闭环控制何时介入、采取了什么动作

- `GRASP_RECENTER`
  - 是否发生抓取回正

- `GRASP_SUCCESS / GRASP_FAILED`
  - 抓取成败

- `PLACE_FINISH`
  - 放置流程结束

### 可直接用于实验报告的指标

- 视觉误差
- 抓取成功率
- 抬升后 `grasp_confidence`
- 抬升后 `slip_risk`
- 放置前 `pressure_lateral_ratio`
- 闭环触发次数
- recenter 触发次数

## 8. 论文/答辩可用的一句话概括

本系统构建了一个基于 PyBullet 的机械臂多模态抓取框架，融合视觉、压力与惯性信息实现抓取状态评估，并通过闭环夹持调节与抓取回正机制，提高了仿真抓取流程的稳定性与鲁棒性。
