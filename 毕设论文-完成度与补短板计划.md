# 毕设论文完成度与补短板计划

## 1. 当前项目一句话定位

本项目已经完成了一个**基于 PyBullet 的 Panda 机械臂多模态抓取仿真系统**，能够融合**视觉、压力、惯性**信息评估抓取状态，并通过**闭环夹持调节**与**抓取回正（recenter）**提升抓取与搬运稳定性。

## 9. 已补充的批量实验脚本

当前仓库已新增：

- `scripts/batch_experiments.py`
- `scripts/analyze_batch_results.py`

用途：

- 批量运行抓取实验
- 自动导出 CSV
- 统计抓取成功与放置完成情况
- 记录视觉误差、抓取置信度、滑移风险、闭环控制次数、recenter 次数等指标

### 基本用法

在项目根目录执行：

```bash
python scripts/batch_experiments.py --runs 20
```

### 常用参数

- `--runs 20`：运行 20 次
- `--output logs/batch_experiments.csv`：指定 CSV 输出文件
- `--log-file logs/grasp_system_batch.log`：指定批量实验日志
- `--base-seed 1000`：设置起始随机种子
- `--disable-place`：只做抓取与抬升，不做放置
- `--disable-closed-loop`：关闭闭环控制，用于对比实验
- `--disable-recenter`：关闭抓取回正，用于对比实验
- `--enable-refine-pass`：启用视觉二次细化
- `--enable-auto-bias-calibration`：启用视觉偏差自动校正
- `--object-urdf xxx.urdf`：指定实验物体

### 推荐的第一组实验

1. 基线实验：

```bash
python scripts/batch_experiments.py --runs 20
```

2. 无闭环对比实验：

```bash
python scripts/batch_experiments.py --runs 20 --disable-closed-loop
```

3. 无回正对比实验：

```bash
python scripts/batch_experiments.py --runs 20 --disable-recenter
```

这三组实验可以直接支撑论文中“闭环控制与回正策略有效性分析”。

### 结果分析与画图

完成批量实验后，可执行：

```bash
python scripts/analyze_batch_results.py --input logs/batch_experiments.csv
```

默认输出目录：

- `logs/batch_analysis/summary.md`
- `logs/batch_analysis/batch_overview.png`
- `logs/batch_analysis/quality_trends.png`

可直接用于：

- 论文实验结果初步整理
- 找出高风险样本
- 对比闭环控制与回正策略效果
